# Unity3DGame

This is a 3d board game modelled after the well known board game Hotel Tycoon. It was produced in the context of a Graphics Course at the Technical University of Crete 

Only the scripts written for the implementation are provided This repository is not meant to be compiled in order to produce a new executional file. It is only meant to provide an idea of the quality of code of this project.

The resulting executional file IS provided.
